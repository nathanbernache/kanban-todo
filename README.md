# Kanban Board

A drag-and-drop Kanban board for managing tasks across three columns: **To Do**, **In Progress**, and **Completed**.

Built with React, Vite, and the HTML5 Drag and Drop API. Data persists in browser localStorage.

## Features

- Create, edit, and delete tasks with title, description, and tags
- Drag and drop tasks between columns
- Tag support with inline add/remove
- Persistent storage via localStorage

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Build

```bash
npm run build
npm run preview
```
